# Tao Te Ching Daily Study

A ChatGPT Skill for studying one chapter of the Tao Te Ching each day.

## Features
- Chinese source text
- Faithful English translation
- Concise Chinese interpretation
- Daily practice
- Reflection question

## Main triggers
- 开始今天的《道德经》学习
- 学习第X章
- 继续上一篇
